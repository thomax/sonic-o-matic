# sonic-o-matic

Crowdsourced Spotify playlist assembly :-)

## Usage

	git clone git@github.com:thomax/sonic-o-matic.git
	cd sonic-o-matic
	bundle install

Get a Spotify app key.
Edit config/spotify_credentials.yml

	./bin/sonic-o-matic start
